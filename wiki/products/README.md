# Products

Long-lived assets and repositories. See [[fse-stack-architecture]] for how the FSE products relate to each other.

## FSE Stack (active development)

- [wolf-blank](wolf-blank/README.md) — FSE base theme. Zero WP default styling, design token slots for child themes.
- [seijaku-fse](seijaku-fse/README.md) — FSE child theme powering wolfthemes.com redesign. Light, editorial, Lexend + electric blue.
- [wolf-store](wolf-store/README.md) — WordPress plugin: theme marketplace backend (OOP PHP + REST API) + React frontend.
- [wolf-blocks](wolf-blocks/README.md) — Reusable Gutenberg blocks (stats, testimonials, pricing). Consumes theme tokens.

## Legacy Stack (ThemeForest — maintained, not extended)

- [themeforest-portfolio](themeforest-portfolio/README.md) — 48 premium WordPress themes. Organized by category with positioning notes.
- [wolf-core](wolf-core/README.md) — Core functions plugin shared across all legacy themes.
- [wolf-visual-composer](wolf-visual-composer/README.md) — WPBakery Page Builder extension for legacy themes.

## Infrastructure & Tools

- [wolfthemes-wiki](wolfthemes-wiki/README.md) — VitePress docs site at wiki.wolfthemes.com.
- [ai-crew](ai-crew/README.md) — Python/CrewAI automation for support and operations.
