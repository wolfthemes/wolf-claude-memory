## Taxonomy notes

This table reflects the optimized taxonomy (June 2026). Key changes from the original ThemeForest state:

- **New categories created:** Fashion, Fitness, Performing Arts, Church/Religious, Art Gallery/Museum, Adventure/Outdoor, Wine, Tech/SaaS, Video/Film, Print-on-Demand, Tattoo/Lifestyle
- **"Shop" and "Other" retired** as catch-all categories — every theme now has a specific vertical
- **Vonzot**: color is Pink (not Blue) — critical for buyer filtering
- **Soundkraft**: color is Red/Black (not Grey) — "deep red stage lighting" per positioning
- **Snakepit accent**: `#7ea8a4` (muted teal/sage) added as Teal alongside Black/White
- **Hyperbent**: removed from portfolio June 2026
- Full positioning rationale per theme: `raw/themeforest-portfolio/theme-positioning.md`
