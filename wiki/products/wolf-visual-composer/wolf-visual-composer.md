# wolf-visual-composer

WPBakery Page Builder extension plugin for WolfThemes premium themes. Registers custom WPBakery elements and extends its functionality to match WolfThemes theme capabilities.

**GitHub:** https://github.com/wolfthemes/wolf-visual-composer  
**Plugin URI:** https://wolfthemes.com/plugin/wolf-visual-composer  
**Current version:** 3.8.0  
**Requires:** WordPress 6.0+

## Purpose

Add WolfThemes-specific content elements (blocks, shortcodes, custom row/column options) to WPBakery Page Builder. Bundled or required alongside legacy ThemeForest themes that support WPBakery.

## Relationship to the FSE stack

Belongs to the **legacy Elementor/WPBakery stack**. The new FSE projects (wolf-blank, seijaku-fse, wolf-store, wolf-blocks) replace this approach entirely with native Gutenberg blocks. No new development planned for this plugin.
